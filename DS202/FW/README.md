# 文件说明

固件为hex文件，文件名由三个字段组成，如: 
```
DS202_APP2_v15.6.6.hex
  |    |     |____字段3：版本号
  |    |__________字段2：APP1|APP2
  |_______________字段1：固件名称
```
1. 字段1：表示DS202固件。
2. 字段2：可选项为APP1或者APP2。DS202可以同时写入两个固件，默认开机执行APP1，
   按 >|| 键开机选择APP2执行。官方固件使用APP1，测试固件建议选择APP2.
3. 字段3：版本信息。三个数字为固件发布日期，可在DS202 Product菜单下查看固件版本。

注意：写入固件时请修改hex文件名到 *8个大写字符*以下，然后进入DFU，拷贝到虚拟U盘升级APP

# 修订历史

## v15.6.12

1. 重写内部FLASH校准数据存储代码，结构更精简，内部相关FLASH寿命提高16倍。
2. 优化待机锁屏机制，待机以后屏蔽触摸板和运行按键，只能短按电源键解锁，防止待机中误触。
3. 增加手动进入待机模式，即：长按电源键弹出关机菜单，然后短按电源键直接进入待机。
   待机过程中LED闪烁，短按电源键恢复显示。目前待机后MCU仍然全速运行，后期优化空间巨大。
4. 优化CSV存储功能，填充数据由0x00改为空格0x20，文本文件中不再有乱码。
   CSV保存的数据由屏幕纵向坐标改为实际电压值，范围-100V到100V，分辨率1mV，数据参考如下：
```    
    CH1,    CH2,
 -2.800, -0.320,
 -2.240, -0.080,
 -2.000, -0.080,
 -1.680, -0.080,
 -1.440, -0.080,
 -1.040, -0.080,
...
```

## v15.6.7

1. 修复System Setting项目下MenuCycle、ItemCycle设置无效问题。复位值默认开启。
2. CH_A、CH_B复位值改为DC耦合。

## v15.6.6

初始发布，基于官方v1.28，主要改动如下：

1. 电池电压菜单上单击弹出电池信息子菜单，可以显示电池实时电压、电流等信息。
2. 增加主菜单隐藏时CH_B电压档快速调整操作。选中主菜单第一项中的CH_B，然后隐藏主菜单，
   上下滑动时修改CH_B电压档，否则上下滑动修改CH_A电压档。
